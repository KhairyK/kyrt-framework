# Contributing to KYRT Framework

Terima kasih sudah ingin berkontribusi! ❤️

## Langkah-langkah
1. Fork repository KYRT Framework
2. Buat branch baru: `git checkout -b fitur-baru`
3. Lakukan perubahan / perbaikan
4. Commit perubahan: `git commit -m "Deskripsi perubahan"`
5. Push ke branch-mu: `git push origin fitur-baru`
6. Buat Pull Request di GitHub

## Rules
- Ikuti standar kode & indentasi
- Jangan hapus file utama tanpa alasan
- Gunakan bahasa yang sopan & jelas di commit
